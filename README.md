# Catastrophe-Risk-Modelling-A-Monte-Carlo-Analysis-of-Flood-Losses-Inspired-by-the-2024-Valencia-DANA
A simplified catastrophe risk model that estimates flood losses on a residential insurance portfolio using Monte Carlo simulation. Inspired by the 2024 Valencia DANA, the project analyses loss distributions, key risk metrics (VaR &amp; TVaR), and the impact of Excess of Loss reinsurance.
